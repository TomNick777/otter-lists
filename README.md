# 🦦 海獭待办

一个简约但独特的 GTD 待办清单应用。

## ✨ 特性

- **GTD 工作流** — 收集箱、下一步、项目、等待中、也许，完整的 Getting Things Done 流程
- **彩色卡片** — 每个分类拥有专属配色，视觉清晰
- **进度追踪** — 实时显示完成进度条
- **日历视图** — 切换日历模版，按日期查看任务
- **自定义清单** — 右键重命名、换色、删除，亦可添加新清单
- **本地存储** — 数据保存在浏览器，无需登录

## 🚀 使用

直接打开 `index.html` 即可使用。

或访问在线版本：[https://tomnick777.github.io/otter-lists/](https://tomnick777.github.io/otter-lists/)
